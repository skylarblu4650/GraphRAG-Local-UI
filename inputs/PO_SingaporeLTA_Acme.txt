2. PROJECT OVERVIEW: SMART INTERSECTION & FLEET TELEMATICS (SIFTAN)
The Buyer is initiating the SIFTAN project to deploy automated, low-latency traffic analytics and commercial fleet tracking across 50 major intersections. To minimize bandwidth and ensure strict data privacy, the architecture requires heavy edge-compute capabilities where video feeds and telemetry are processed locally at the intersection, with only anonymized metadata transmitted back to the central LTA command center.

3. SECURITY, COMPLIANCE, AND DATA SOVEREIGNTY
3.1 IM8 Compliance: All hardware and commercial off-the-shelf (COTS) software deployed under this SOW must comply with the Singapore Government's Instruction Manual 8 (IM8) for IT security.
3.2 Data Residency: Contractor guarantees that zero Personally Identifiable Information (PII) or raw video feeds will be processed by, or transmitted to, any public cloud infrastructure.
3.3 Air-Gapped Deployment: All software modules must support fully air-gapped, offline deployment. Telemetry reporting to third-party software vendors is explicitly disabled and prohibited.

4. SCOPE OF PROFESSIONAL SERVICES
Contractor shall provide end-to-end integration, including physical installation of edge nodes, configuration of the machine learning pipelines, and integration with the existing LTA centralized dashboard. Contractor assumes all Tier 1 and Tier 2 service level agreements (SLAs) for 99.99% system uptime over a 24-month period.
